# Security Policy

## Supported versions

Only the latest release on the `main` branch receives security updates.
Please make sure you are running the latest version before reporting.

| Version        | Supported          |
| -------------- | ------------------ |
| Latest release | :white_check_mark: |
| Older releases | :x:                |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use GitHub's
[private vulnerability reporting](https://github.com/foXaCe/dreame-vacuum-card/security/advisories/new),
or email <foxace66@gmail.com>.

Please include:

- A description of the issue and its potential impact
- Steps to reproduce
- Affected version(s)
- Any known mitigation

You can expect an initial response within 7 days. Once confirmed, a fix will be
released as soon as practical depending on complexity.

## Scope

This policy covers the Dreame Vacuum Card frontend code in this repository.
Issues in Home Assistant core or third-party dependencies should be reported to
their respective maintainers.
